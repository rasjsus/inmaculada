# Jesucristo unico salvador del mundo

```bash
Jesucristo, único Salvador
del mundo, alimento para la
vida nueva. (Bis) 

Jesucristo tú dijiste:
Yo soy el pan de vida,
que ha bajado del cielo.
El que coma de este pan,
vivirá, vivirá para siempre. 

Y el pan que yo les voy a dar
es mi carne para que el mundo tenga vida
El que coma de este pan,
vivirá, vivirá para siempre. 

```
